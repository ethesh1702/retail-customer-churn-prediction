# Retail Customer Churn Prediction

Machine learning analysis project for predicting retail customer churn using customer, transaction, campaign, support, interaction, and review data.

## Contents

- `Project_Classification_Model.ipynb` - classification modeling notebook
- `customers.csv` / `customers - Copy.csv` - customer profile data
- `transactions.csv` / `transactions - Copy.csv` - transaction history
- `customer_reviews_complete.csv` / `customer_reviews_complete - Copy.csv` - review data
- `interactions.csv` / `interactions - Copy.csv` - customer interaction records
- `support_tickets.csv` / `support_tickets - Copy.csv` - support ticket records
- `campaigns.csv` / `campaigns - Copy.csv` - campaign engagement data
- `Retail Customer Churn Prediction.docx` / `Retail Customer Churn Prediction - Copy.docx` - project report
- `Retail-Customer-Churn-Prediction-A-Machine-Learning-Analysis.pptx` - presentation deck

## Getting Started

Open `Project_Classification_Model.ipynb` in Jupyter Notebook, JupyterLab, VS Code, or Google Colab to review and run the analysis.
